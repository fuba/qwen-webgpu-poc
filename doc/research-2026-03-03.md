# 2026-03-03 調査メモ: Qwen3.5-0.8B を WebGPU でブラウザ実行

## 結論

- `Qwen/Qwen3.5-0.8B` は元モデル配布で、ブラウザ直接実行に必要な ONNX 構成がない。
- `onnx-community/Qwen3.5-0.8B-ONNX` は WebGPU 実行に必要な `onnx/*.onnx` と tokenizer 群を持つ。
- `huggingface/transformers.js` 側には `qwen3_5` 実装が存在する。
- ただしマッピング上、`qwen3_5` は `AutoModelForImageTextToText` 側の扱いが中心で、PoCはこれを採用。
- Simon Willison の local-folder 手法 (`env.customCache`) はこのPoCへ移植可能。
